Tic-Tac-Toe Game

A simple console-based Tic-Tac-Toe game written in C++ using Code::Blocks. This game allows two players to play against each other on the same computer. It demonstrates basic C++ programming concepts such as loops, conditionals, functions, and arrays.

Features

Classic 3x3 Tic-Tac-Toe board

Two-player mode (no AI)

Input validation to prevent invalid moves

Detects win, loss, or draw

How to Play

Compile the project in Code::Blocks or your preferred C++ IDE.

Run the program.

Players take turns entering their move coordinates.

The game announces the winner or a draw at the end.

Purpose

This project is ideal for beginners to practice C++ fundamentals and understand simple game logic.
